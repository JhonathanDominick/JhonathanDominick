## 🚀 Projeto em Destaque: Sistema de Agendamento (Microserviços)

Arquitetura completa de microserviços desenvolvida para consolidar minha transição para Backend Java.

### 🏗️ Arquitetura
Cliente
↓
BFF-AGENDADOR (8086) ← Orquestração & JWT
↓
├── CADASTRAUSUARIO (8083) → PostgreSQL
├── AGENDADOR-TAREFAS (8081) → MongoDB
└── NOTIFICACAO (8085) → E-mail/Thymeleaf
plain
Copy

### ✨ Funcionalidades
- ✅ **Autenticação JWT** stateless entre serviços
- ✅ **BFF Pattern** - Backend for Frontend
- ✅ **Comunicação** via OpenFeign (REST)
- ✅ **Multi-banco:** PostgreSQL + MongoDB
- ✅ **Containerização** completa com Docker Compose
- ✅ **Documentação** automática com Swagger/OpenAPI

### 🔄 Fluxo de Uso
1. Cadastro de usuários com consulta de CEP externo
2. Login com geração de token JWT
3. CRUD completo de tarefas agendadas
4. Notificações automáticas por e-mail
